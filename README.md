# Козел 32

Сетевой MVP карточной игры на 32 карты с комнатами и realtime-логикой.

## Запуск локально

```bash
npm install
npm run dev
```

По умолчанию клиент стартует на Vite-порту, сервер на `http://localhost:3001`. Если нужны другие порты, можно запускать сервер и клиент отдельно через `dev:server` и `dev:client`.

## Что реализовано

- Комнаты по ссылке и realtime через Socket.IO.
- 2-4 игрока.
- Колода 32 карты: 9, валет, дама, король, 10, туз.
- Козырь всегда бубны.
- Каждый игрок видит только свои карты.
- Ход одной или несколькими картами одной масти.
- Ответ: побить карты или сбросить столько же карт рубашкой вверх.
- Сброшенные карты скрыты до конца раунда.
- Банк скрыт во время раунда и раскрывается только в итогах.
- Короткая фаза показа взятки перед уходом карт в банк.
- Добор до 4 карт после взятки, пока есть колода.
- Подсчет очков банка и штрафов.
- Выбывание на 12 штрафных баллах.
- Спец-комбинации: четыре девятки, четыре туза, четыре козыря.
- Анимированные карты, стол и экран результата.

## Render

Проект можно деплоить как один Node Web Service.

```text
Build Command: npm install && npm run build
Start Command: npm start
```

Environment:

```text
NODE_ENV=production
```

Render задает `PORT` автоматически. В production сервер отдает собранный `dist`, а клиент подключается к Socket.IO на том же домене.

## Ограничения

- Состояние хранится в памяти процесса.
- После перезапуска сервера комнаты сбрасываются.
- Если игрок отключился во время игры, текущий раунд останавливается и комната возвращается в лобби.
